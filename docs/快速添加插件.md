# 快速添加插件指南

## 三步添加插件

### 步骤 1: 创建插件文件

在 `src/plugins/` 目录下创建新的 Python 文件，例如 `my_plugin.py`：

```python
"""
我的插件
"""
from nonebot import on_command
from nonebot.adapters.onebot.v11 import MessageEvent
from src.utils.logger import get_logger

logger = get_logger("my_plugin")

# 创建命令
my_cmd = on_command("mycmd", priority=10, block=True)

@my_cmd.handle()
async def handle_my_cmd(event: MessageEvent):
    """处理命令"""
    await my_cmd.send("这是我的插件！")
```

### 步骤 2: 启用插件

#### 方式 A: 通过 SQLite 数据库（推荐）

```bash
# 进入项目目录
cd nonebot-chatbot

# 使用 sqlite3 命令行工具
sqlite3 data/bot.db

# 在 SQLite 中执行
INSERT INTO plugin_configs (plugin_name, enabled, config_data) 
VALUES ('my_plugin', 1, '{}');

# 退出
.quit
```

#### 方式 B: 通过 Python 脚本

创建临时脚本 `enable_plugin.py`：

```python
import asyncio
from src.database.connection import DatabaseManager
from src.database.plugin_config import PluginConfig
from config import load_config

async def enable_plugin():
    config = load_config()
    db_manager = DatabaseManager(config.database)
    await db_manager.init_db()
    
    async with db_manager.get_session() as session:
        await PluginConfig.create_or_update(session, "my_plugin", enabled=True)
        print("插件已启用")

asyncio.run(enable_plugin())
```

运行：
```bash
python enable_plugin.py
```

### 步骤 3: 重启机器人

```bash
python bot.py
```

插件会自动加载并启用！

## 常用命令模板

### 简单命令

```python
from nonebot import on_command
from nonebot.adapters.onebot.v11 import MessageEvent

cmd = on_command("命令名", priority=10, block=True)

@cmd.handle()
async def handle_cmd(event: MessageEvent):
    await cmd.send("响应消息")
```

### 带参数的命令

```python
from nonebot import on_command
from nonebot.params import CommandArg
from nonebot.adapters.onebot.v11 import MessageEvent, Message

cmd = on_command("命令名", priority=10, block=True)

@cmd.handle()
async def handle_cmd(event: MessageEvent, args: Message = CommandArg()):
    arg_str = str(args).strip()
    await cmd.send(f"收到参数: {arg_str}")
```

### @机器人触发

```python
from nonebot import on_message
from nonebot.rule import to_me
from nonebot.adapters.onebot.v11 import MessageEvent

handler = on_message(priority=10, block=True, rule=to_me())

@handler.handle()
async def handle_msg(event: MessageEvent):
    await handler.send("你@我了！")
```

```bash
grep "my_plugin" logs/bot.log
```

```bash
sqlite3 data/bot.db

# 查看所有插件配置
SELECT * FROM plugin_configs;

# 查看插件状态
SELECT * FROM plugin_status WHERE plugin_name = 'my_plugin';
```

## 常见问题

### Q: 插件没有加载？

**检查清单：**
1. ✅ 文件是否在 `src/plugins/` 目录
2. ✅ 文件名是否正确（不能以 `_` 开头）
3. ✅ 插件是否在数据库中启用
4. ✅ 查看日志是否有错误

### Q: 如何调试插件？

1. 查看日志：`logs/bot.log`
2. 添加调试日志：
   ```python
   logger.debug("调试信息")
   ```
3. 检查命令优先级和 block 参数

### Q: 插件如何访问数据库？

参考 `src/plugins/example_plugin.py` 或查看完整文档：`docs/插件开发指南.md`
