## 测试执行方式

### 快速测试

```bash
# 运行所有测试
pytest tests/

# 运行特定模块
pytest tests/test_config.py
pytest tests/test_database.py
```

### 详细测试

```bash
# 显示详细输出
pytest tests/ -v

# 显示覆盖率
pytest tests/ --cov=src --cov=config --cov-report=html
```

### 使用测试脚本

```bash
# Linux/Mac
./tests/run_tests.sh

# Windows
tests\run_tests.bat
```

## 测试环境要求

1. **Python**: 3.9+
2. **依赖**: 
   - pytest
   - pytest-asyncio
   - pytest-cov
   - pytest-mock

安装测试依赖：
```bash
pip install -r requirements.txt
```

---

## 测试最佳实践

1. **运行测试前**: 确保已安装所有依赖
2. **测试数据库**: 使用临时SQLite数据库，测试后自动清理
3. **异步测试**: 使用 `@pytest.mark.asyncio` 装饰器
4. **Fixtures**: 使用 `conftest.py` 中的fixtures避免重复代码
5. **隔离性**: 每个测试独立，不依赖执行顺序
